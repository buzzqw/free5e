# Roboto

These font files were downloaded via the [google-webfonts-helper](https://gwfh.mranftl.com/fonts/roboto?subsets=latin,latin-ext,math,symbols).

The selected styles were:
- regular
- italic
- 700 (bold)
- 700italic (bold + italic)
